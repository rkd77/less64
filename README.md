less64
======

This is a simple text viewer for esxdos for Timex 2048 and compatibles.
less64s is for ZX Spectrum. Font for hires mode has ISO8859-2 codepage.

Usage:
```
.less64 somefile
```

less85
======

It is similar, but with with a 6x8 font, which was based on fonts
from http://int10h.org/oldschool-pc-fonts/

To build:
```
make
```


Keys:
-----

```
space - Page down
b - Page up
g - Go to begining
h - Help
q - Quit
```
